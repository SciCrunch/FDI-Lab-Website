Projects

We partner with national and international initiatives to build open data infrastructure across biomedical research.

Active Projects

BICAN Brain Atlas

The Brain Initiative Cell Atlas Network (BICAN) aims to create a comprehensive cellular atlas of the mammalian brain, including humans, non-human primates, and model organisms. FDI Lab contributes data infrastructure and FAIR standards support.

FDI Lab project page →    Visit BICAN ↗

dkNET Data Portal

dkNET is a data and resource-sharing network for the digestive, kidney, and diabetes research communities. It provides a portal for discovering datasets, tools, and resources to advance NIDDK mission research.

FDI Lab project page →    Visit dkNET ↗

SPARC Autonomic Nervous System

The Stimulating Peripheral Activity to Relieve Conditions (SPARC) program maps the autonomic nervous system to enable development of next-generation neuromodulation devices. FDI Lab supports FAIR data infrastructure and portal development for the SPARC data ecosystem.

FDI Lab project page →    Visit SPARC ↗

ODC-SCI Open Data

The Open Data Commons for Spinal Cord Injury (ODC-SCI) is a repository and portal for sharing spinal cord injury research data. The project aims to maximize the value of existing data through reuse and integration.

FDI Lab project page →    Visit ODC-SCI ↗

ReproNim Reproducibility

ReproNim (Reproducible Neuroimaging) is a Center for Reproducible Neuroimaging Computation. The project develops standards, tools, and training to improve reproducibility in neuroimaging research.

FDI Lab project page →    Visit ReproNim ↗

RRID Resource ID

The Resource Identification Initiative (RRID) provides unique, persistent identifiers for key biological resources — including antibodies, model organisms, cell lines, and software tools — cited in scientific literature. RRIDs dramatically improve reproducibility and citability of research resources.

FDI Lab project page →    Visit RRID ↗

TARA Acupuncture Research

The Topological Atlas and Repository for Acupuncture Research (TARA) establishes a comprehensive resource for the acupuncture research and clinician community, including an acupoint ontology, MRI-based human atlases, and a searchable acupoint database.

FDI Lab project page →    Visit TARA ↗

NIF Neuroscience

The Neuroscience Information Framework (NIF) — the precursor to FDI Lab — created the first cross-database search engine for neuroscience resources and established the NeuroLex ontology for neuroscience. NIF infrastructure continues to be maintained and expanded.

FDI Lab project page →    Visit NIF ↗

Past Projects

BICCN Brain Cell Census

The Brain Initiative Cell Census Network (BICCN) created a comprehensive census of cell types in the mouse, human, and marmoset brain. FDI Lab contributed data standards, ontologies, and portal infrastructure to make this atlas openly accessible.

FDI Lab project page →    Visit BICCN ↗

CINERGI Geoscience

Community Inventory of EarthCube Resources for Geosciences Interoperability (CINERGI) created a catalog and discovery system for geoscience data resources, applying the SciCrunch platform to Earth sciences.

FDI Lab project page →

D3R Drug Discovery

The Drug Design Data Resource (D3R) hosts community-wide grand challenges in computer-aided drug design and provides curated datasets to advance structure-based drug design methodologies.

FDI Lab project page →    Visit D3R ↗

FSCI Training

The FORCE11 Scholarly Communication Institute (FSCI) is an annual summer program offering advanced training in open science, data management, and scholarly communication. FDI Lab faculty lead and contribute to FSCI courses.
